# 中文版
需要翻译如下文件：
* `config/article.js`
* `config/onward-journey.js` 加入六组推荐文章显示在末尾
* `views/layout.html` 中HTML标签中的文字。
* `client/index.html` 中的文字。

注意，HTML文件中的`{% %}`和`{{ }}`标签不要动。
